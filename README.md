# Hyper 一个 Vue UI组件库

在安装使用本框架的时候，请在 CSS 中设置 box-sizing 为 border-box：

```css
*{box-sizing: border-box}
:root{
    --button-height:32px;
    --font-size:14px;
    --button-bg:white;
    --button-active-bg:#eee;
    --border-radius:4px;
    --color:#333;
    --border-color:#999;
    --border-color-hover:#666;
}
```

引入组件时，要引入相关CSS：

`import 'hyper-demo-0.0.1/dist/index.css'`
